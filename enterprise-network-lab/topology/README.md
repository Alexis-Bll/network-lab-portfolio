# Topology

## Overview

This directory contains the network topology diagrams for the Enterprise Network Lab.

Two versions of the lab are included to demonstrate progression from foundational networking concepts to a more realistic enterprise design.

---

## Versions

### V1 – Core Enterprise Design

The first version of the lab focuses on core CCNA-level networking concepts, including:

- VLAN segmentation
- Inter-VLAN routing
- HSRP (First Hop Redundancy)
- OSPF (dynamic routing)
- Rapid Spanning Tree Protocol (RSTP)
- EtherChannel (LACP)
- Basic Layer 2 and Layer 3 design

This version represents a solid internal enterprise network without external connectivity.

---

### V2 – Enhanced Enterprise Network

The second version builds upon V1 and introduces additional real-world elements to better simulate a production environment:

- Integration of a Private MPLS WAN connecting HQ and branch sites
- Addition of Public Internet access via an ISP edge
- Implementation of NAT (Network Address Translation)
- Introduction of centralised services (e.g. DHCP, DNS, Active Directory via DC1)
- Inclusion of an Application Server
- Improved realism in network structure and traffic flow

This version reflects how enterprise networks interact with both private WAN infrastructure and external networks.

---

## Purpose

The purpose of maintaining two topology versions is to clearly demonstrate:

- Progression in networking knowledge
- Ability to evolve and redesign a network
- Understanding of both internal and external network architecture

---

## Notes
- Diagrams are created using draw.io
- V2 represents the current and more complete implementation
- Further versions may be added as additional technologies are explored
