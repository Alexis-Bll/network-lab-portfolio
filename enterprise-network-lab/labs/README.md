# Lab Documentation

This folder contains versioned documentation for the enterprise network lab.

The lab has been structured to reflect iterative development, similar to how real-world network environments evolve over time.

---

## Versions

### v1 – Core Enterprise Lab (Completed)

This version demonstrates a complete CCNA-level enterprise network, including:

- Base device configuration
- Core and access layer setup
- VLANs and trunking
- Inter-VLAN routing (SVIs)
- OSPF routing (multi-site WAN)
- HSRP redundancy
- EtherChannel (LACP)
- ACLs for traffic control
- Layer 2 security (Port Security, DHCP Snooping, DAI)
- Testing and validation

📂 Path: `labs/v1/`

---

### v2 – Enterprise Services Expansion (In Progress)

This version extends the lab to include more real-world enterprise features:

- Internet connectivity
- NAT (inside/outside)
- DHCP (centralised)
- Active Directory and DNS services
- Additional client devices (dynamic addressing)
- More realistic user segmentation

📂 Path: `labs/v2/`

---

## Purpose

The goal of versioning this lab is to:

- Show progression in technical ability
- Demonstrate real-world network evolution
- Build a more complete enterprise environment beyond CCNA scope
