# Device Configurations

This folder will contain the final running configurations for all network devices in the enterprise lab.

Each configuration file will represent the completed setup of routers, switches, and other devices after all lab phases have been implemented.

---

## Current Status

🚧 Configurations are currently being developed and tested as part of the lab build process.

At this stage, the focus is on:

* Designing the network topology
* Implementing core Layer 2 and Layer 3 functionality
* Validating connectivity and redundancy

Finalised configurations will be uploaded once the full lab has been completed and verified.

---

## What Will Be Included

* Core switch configurations (VLANs, STP, HSRP, EtherChannel)
* Routing configurations (OSPF)
* Access layer switch configurations
* Branch router configurations
* Security configurations (ACLs)

## Notes

This repository is being actively updated as I progress through building a full enterprise network simulation.

The goal is to ensure all configurations are clean, tested, and production-ready before publishing.
