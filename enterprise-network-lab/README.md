# Enterprise Network Lab

## Overview

This lab simulates a small enterprise network using technologies covered in the Cisco CCNA.

The goal of the lab is to combine multiple networking concepts together into a realistic network environment.

## Technologies Used

- VLANs
- Inter-VLAN Routing
- OSPF
- HSRP
- EtherChannel
- NAT
- Access Control Lists (ACL)

## Lab Environment

The lab is built using:

- EVE-NG
- Cisco IOS routers and switches
- Virtual PCs for testing

## Objectives

- Implement VLAN segmentation
- Configure inter-VLAN routing
- Deploy OSPF dynamic routing
- Implement HSRP for gateway redundancy
- Configure NAT for internet access
- Apply ACLs for traffic filtering
