# Network Lab Portfolio

This repository contains my personal networking labs and projects built using **EVE-NG**.

The purpose of this repository is to document practical networking scenarios and demonstrate my hands-on experience with technologies covered in the **Cisco CCNA** and **Fortinet NSE4** certifications.

---

## About Me

I am an aspiring **Network Engineer from the UK** with a **First Class degree in Forensic Computing & Security from Bournemouth University**.

I have achieved the **Cisco CCNA certification** and I am currently studying for the **Fortinet NSE4 certification**.

My goal is to build a career in **network engineering, infrastructure and network security**.

---

## Lab Environment

My labs are built using:

- **EVE-NG** for network simulation
- **Cisco IOS routers and switches**
- **Fortinet firewall images**
- **Virtual PCs and servers**

This allows me to simulate real-world enterprise network environments.

---

## Topics Covered

The labs in this repository demonstrate concepts including:

- VLAN configuration
- Inter-VLAN routing
- OSPF dynamic routing
- HSRP gateway redundancy
- EtherChannel link aggregation
- NAT for internet access
- Access Control Lists (ACL)
- Firewall configuration
- VPN technologies
